# Casa Reserva da Serra — Orçamento (web)

Visualizador publicado do orçamento da Casa Reserva da Serra (Stemmer 337-01 Estudo 04, Canela/RS).

**Acesso protegido por senha.** Apenas o conteúdo encriptado client-side (AES-256-GCM via PBKDF2) está neste repo.

Fonte: repo privado [`tiagofreitas83/337-01-orcamento`](https://github.com/tiagofreitas83/337-01-orcamento).

Para atualizar: regenerar `orcamento.html` no repo privado → `npx staticrypt orcamento.html -p '<senha>' --short -d build` → copiar `build/orcamento.html` para `index.html` aqui → push.
